# 宠物商城项目简介 java842

## 项目概述

宠物商城是一个基于 Spring Boot 和 Vue 的电子商务平台，旨在为用户提供一个便捷、高效的宠物用品购物环境。

## 技术栈

### 后端

- **语言**: Java
- **框架**: Spring Boot + MyBatis
- **数据库**: MySQL

### 前端

- HTML
- CSS
- JavaScript
- jQuery

## 功能模块

系统主要分为以下功能模块：

- 用户模块
- 商铺模块
- 管理员模块

每个模块都包含了相关角色所需的功能点。

## 系统角色

系统定义了以下三个角色：

- **管理员**: 负责整个商城的运营管理。
- **用户**: 在商城中购买商品的普通用户。
- **商铺**: 在商城内销售宠物用品的商家。

## 运行环境

- 开发工具: IntelliJ IDEA

## 部署步骤

1. 配置好开发环境（IDEA + Java + MySQL）
2. 导入项目代码
3. 初始化数据库
4. 启动后端服务
5. 构建前端项目
6. 部署到服务器
